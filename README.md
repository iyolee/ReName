# reset-ext

> reset-ext is a NPM package can batch change file extensions.

### Installation

```
npm install reset-ext -g
```

### Usage

check version

```
reset-ext -v
```

batch file

```
reset-ext [path] [prevFilename] [newFilename]
```

### Example

```
reset-ext . index.js index.ts
reset-ext ./example .js .ts
```
